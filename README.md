# Mixs.ts

MVC library for faster development for typescript

### Let's start
#### Installation
```
npm install mixsts
```
init typescript tsconfig.json Note: make sure you installed typescript<br>
```
tsc --init
```
We need to do some configuration in **tsconfig.json**
### Folder's structure
```
libs
│   app.ts // contain server file
└───configs
│   │   app.ts
└───controllers
    │   app.controller.ts
```
